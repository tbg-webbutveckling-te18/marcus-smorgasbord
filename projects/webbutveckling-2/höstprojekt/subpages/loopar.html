Loopar

En loop används för att loopa igenom en t.ex. en lista med olika värden eller ord. Man kan göra så den loopar tills den når ett visst värde eller tills man stopar den. finns väldigt mycket man kan göra med loopar och detta var bara ett exempel

JavaScript For Loop

Sidan börjar med att göra en lista som inte har någon text is sig. Sedan så har den en till lista som heter i som börjar med värdet 0. Vi har en for loop som säger att om värdet är 0 så ska sidan plusa på 1 ända tills värdet är mindre än 5.

Så hära ser koden ut: var text = ""; var i; for (i = 0; i < 5; i++) { text += "The number is " + i + "
"; }
