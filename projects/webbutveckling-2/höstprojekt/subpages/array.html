Array

En Array är som en byrå med många lådor man kan spara saker i. Man kan alltså göra en Array med många variabler i för att ha lite mera struktur på saker. När man vill andropa variabeln så skriver man var den ligger i arrayen/listan. Man kan även spara flera arrays i andra arrays.

JavaScript Arrays

Här är det en väldigt enkel lista med bilmärken. Dom hära bilmärkerna är sparad i en array så när jag anropar på min array(cars) så kommer allt i listan visas.
